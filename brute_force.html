#################################################################

		Bruteforce Algorithm (Naive Algorithm)

#################################################################


The most basic of all exact string search algorithms would be the
naive algorithm where each element of pattern to be searched is 
compared with the elements of the text in which pattern is to be
searched, to decide if a match occurs or not. Suppose the following
strings,

Text = AATGTACTAGAGTACT
Pattern = GTAC

The algorithm proceeds as follows:

	---------------------------------
	|     x				|
	|     |  = character mismatch	|
	|				|
	|     /				|
	|     |  = character match	|
	|				|
	---------------------------------


	AATGTACTAGAGTACT
	x
	|
	GTAC

****************************************************************

	AATGTACTAGAGTACT
	 x
	 |
	 GTAC

****************************************************************

	AATGTACTAGAGTACT
	  x
	  |
	  GTAC

****************************************************************

	AATGTACTAGAGTACT
	   /
	   |
	   GTAC

****************************************************************

	AATGTACTAGAGTACT
	   //
	   ||
	   GTAC

****************************************************************

	AATGTACTAGAGTACT
	   ///
	   |||
	   GTAC

****************************************************************

	AATGTACTAGAGTACT
	   ////
	   ||||
	   GTAC

					(Pattern found)

****************************************************************

	AATGTACTAGAGTACT
	    x
	    |
	    GTAC

****************************************************************

	AATGTACTAGAGTACT
	     x
	     |
	     GTAC

****************************************************************

	AATGTACTAGAGTACT
	      x
	      |
	      GTAC

****************************************************************

	AATGTACTAGAGTACT
	       x
	       |
	       GTAC

****************************************************************

	AATGTACTAGAGTACT
	        x
	        |
	        GTAC

****************************************************************

	AATGTACTAGAGTACT
	         /
	         |
	         GTAC

****************************************************************

	AATGTACTAGAGTACT
	         /x
	         ||
	         GTAC

			(Second character of pattern mismatched)

****************************************************************

	AATGTACTAGAGTACT
	          x
	          |
	          GTAC

****************************************************************

	AATGTACTAGAGTACT
	          x
	          |
	          GTAC

****************************************************************

	AATGTACTAGAGTACT
	           /
	           |
	           GTAC

****************************************************************

	AATGTACTAGAGTACT
	           //
	           ||
	           GTAC

****************************************************************

	AATGTACTAGAGTACT
	           ///
	           |||
	           GTAC

****************************************************************

	AATGTACTAGAGTACT
	           ////
	           ||||
	           GTAC

				(Match found)

****************************************************************

	AATGTACTAGAGTACT
	            x
	            |
	            GTAC

				(Procedure complete)

****************************************************************

Let n = length of the text string
m = length of the pattern string

In this, we compare first element of text with the first element of
the pattern. If match does not occur, compare subsequent elements of
text with first element of the pattern till a match occurs or till
element n-m has been compared.
In case a match occurs, compare subsequent elements till a mismatch 
occurs or whole of the pattern gets matched. If somewhere in between,
a mismatch occurs, continue with comparing the first element of the
pattern with the element next to the element of text last compared
with the first element of pattern. If no mismatch occurs and end of
pattern has been encountered, return index of the element of text
which was last compared with first element of pattern.

In worst case, algorithm runs in quadratic time, which is O(mn).
